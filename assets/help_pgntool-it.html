Esporta il database di gioco nella sd-card

Tutte le partite nel database di gioco possono essere esportate nella sd-card in un singolo file PGN. Questo file verrà denominato chess.pgn e un eventuale file esistente verrà sovrascritto.

Importa le partite nel database di gioco

Usa questa opzione per aggiungere partite al database di gioco. L'opzione permette di navigare fino ad un file PGN. Seleziona e importa.

Cancella tutte le partite nel database di gioco

Usa questa opzione se vuoi ripulire il database di gioco. Prima assicurati di aver fatto un backup sulla sd-card, poiché l'azione non può essere annullata.

Crea openingdatabase

Naviga fino ad un file PGN sulla sd-card e questo verrà convertito in un openingdatabase con lo stesso nome ma con un nome file con estensione .bin. Attualmente tutte le partite nel file PGN saranno analizzate fino al 17 ply.

Imposta openingdatabase

Naviga fino all'appena creato file .bin openingdatabase per permettere a Chess di utilizzarlo.

Installa motore UCI

Naviga fino al file binario del motore UCI per permettere a Chess di utilizzarlo.

Disinstalla motore UCI

Elimina il motore UCI installato.

Crea pratica set

Crea posizioni per la pratica dalle partite terminate con uno scacco matto. Naviga fino al file per elaborarlo. Le partite respinte normalmente lo sono perché non finiscono in uno scacco matto o sono dei duplicati

Azzera il set di pratica

Vuoi ricominciare la pratica? Questa opzione azzererà anche il contatore del tempo medio.
